# Игра "Змейка"

## О проекте
Цель проекта - получение опыта разработки в стиле ООП  
Написана в двух вариантах, с использованием Canvas API и с использованием DOM API   
Для переключения между ними см. файл `index.html`

## Управление
Используйте стрелки для управления змейкой, пробел для постановки игры на паузу

## Запуск
```sh
git clone https://github.com/David-Vanuxin/snake-dom-api.git &&
cd snake-dom-api &&
npm i serve &&
npx serve
```

После запуска перейдите по ссылке, которую увидите в терминале
